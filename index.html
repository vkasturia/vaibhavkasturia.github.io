Project Experience
  • Query Understanding via Entity Linking
    • Goal: Understanding the meaning behind search engine queries using entity linking to show more relevant results to the user.
    • Answer times of web search engines typically are below several hundreds of milliseconds; entity linking should thus ideally not consume more than some tens of milliseconds.
    • Developed an entity linking tool achieving higher accuracy at a faster speed compared to several existing entity linking tools.
  • Total Recall in Systematic Reviews
    • Total Recall: Finding all the relevant publications on a given topic.
    • "Recall" of the system: The portion of relevant documents that a search system returns from the set of all relevant documents.
    • Goal: Find all relevant documents, given a collection of several 1000s of documents somewhat related to a specified topic.
    • Implemented several ML methods available in HiCAL on botanical research datasets.
  • Argumentative Axiomatic Re-Ranking for Medical Search Queries
    • People use search technologies to seek health advice online
    • Time-pressured clinicians rely on search results to decide upon the best treatment/diagnosis/test for a patient.
    • Problem: Using search engines in order to complete such decision making tasks, users are able to discern authoritative from unreliable information and correct from incorrect information.
    • Solution: Re-rank search results obtained by traditional search models, in order to promote more argumentative results for keyword-based queries that might benefit from argumentative answers.
    • Participated with the axiomatic re-ranking approach at the TREC 2019 Conference.

Activities

Alexandria Project (ERC Advanced Grant Project)
  • Building and Querying Semantic Layers for Web Archives
    • Exploring Web Archives (huge archived web page collections) by extracting Semantic Profiles or Layers from these collections.
    • Semantic Layers allow describing metadata information about the archived documents, annotating them with useful semantic information, and publishing this data on the Web as Linked Data.
    • Such structured repositories offer advanced query and integration capabilities and make web archives directly exploitable by other systems and tools.
  • Ranking Archived Documents for Structured Queries on Semantic Layers
    • Results returned by structured queries on Semantic Layers are numerous and moreover they all equally match the query.
    • Formalized the task of ranking archived documents for structured queries on Semantic Layers.
    • Proposed two ranking models which jointly consider: i) the relativeness of documents to entities, ii) the timeliness of documents, and iii) the temporal relations among the entities.
